# vinned
Find car details using the vin and/or plate number

#INSTRUCTIONS FOR COLLOBORATORS
*Set up your virtual environment. I recommend **virtualenv and virtualenvwrapper**
*in you virtual env install django, tailwindcss (might change this later) and Postgresql-14.
*create a .env file in your IDE and make sure to add it to git ignore and then add all your keys
making sure to match the variables names in the project
